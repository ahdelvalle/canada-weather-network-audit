üìã Methodology

Each audit systematically tests all registered weather stations via Environment Canada's Historical Climate Data API (climate.weather.gc.ca) and cross-references against the official station inventory from ftp.tor.ec.gc.ca.

Stations are classified as ACTIVE if they return precipitation data within the test period, INACTIVE/NO_DATA if API requests return empty results.
